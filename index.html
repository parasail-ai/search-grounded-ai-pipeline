How It Works

This demo is powered by two APIs: You.com Web Search for live web intel and Parasail for inference on open-weight models. Each section below shows what the API does, how to call it, and what it costs.

▾
🌐 You.com Web Search Powers Open-Weight Models
Open-weight models know the world up to their training cutoff · You.com Web Search fills the gap with live web intel

Models like GPT-4 and Claude are proprietary; you access them only through the vendor's API. Open-weight models (Llama, Qwen, DeepSeek, Gemma) have publicly released weights, so inference providers like Parasail can host and serve them through an OpenAI-compatible API, with full model choice at a fraction of the cost. But they share one hard constraint: a knowledge cutoff. Training ends on a fixed date, and everything after it is a blank. You.com Web Search is what fills that gap, fetching live web results at query time so the model can reason about the world as it is today, not as it was when training stopped.

You.com APIs
Search API · Research API · live web intel · cited answers
Search API
GET ydc-index.io/v1/search · $0.005 / call

Returns live news + web results in a single call — no scraping, no stale index. Used in this demo to ground every LLM step in fresh company signals. Add livecrawl=web to get full page markdown, not just snippets.

import requests resp = requests.get( "https://ydc-index.io/v1/search", headers={"X-API-Key": YDC_API_KEY}, params={ "query": "Salesforce AI news 2026", "num_web_results": 5, "livecrawl": "web", # optional — full page markdown "livecrawl_formats": "markdown", }, timeout=20, ) data = resp.json() news = data["results"]["news"] # [{title, description, url, page_age}, ...] web = data["results"]["web"] # [{title, url, snippets, contents.markdown}, ...]

Research API
Research API
POST api.you.com/v1/research · from $0.012 / call

Goes beyond raw results — runs multiple searches autonomously, reads through sources, and synthesizes a cited, natural-language answer. You get a markdown response with inline citations [[1, 2]] plus the source list. Depth is controlled by the research_effort tier.

import requests resp = requests.post( "https://api.you.com/v1/research", headers={"X-API-Key": YDC_API_KEY}, json={ "input": "What are the latest AI infrastructure trends in enterprise sales?", "research_effort": "standard", # lite | standard | deep | exhaustive }, ) data = resp.json() answer = data["content"] # markdown with inline citations [[1]] sources = data["sources"] # [{url, title, snippet}, ...]
